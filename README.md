# The Cupcake Store

UI para ChartMuseum. Aplicación en **React** y **TypeScript**, empaquetada con **Vite**.

## Requisitos

- Node.js **20** (recomendado: `nvm use` con `.nvmrc`)

## Comandos

```bash
npm install
npm run dev
```

- `npm run build` — compilación para producción
- `npm run preview` — vista previa del build
- `npm run lint` — ESLint
